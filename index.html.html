‚öñÔ∏è Part 4: Interest Rate Parity (IRP)

The $64,000 Question: Which scenario actually happens?

The Core Principle

"In equilibrium, the total expected return from investing in any country should be equal (after adjusting for exchange rate changes)."

Expected Future Spot = Current Spot √ó (1 + Foreign Rate) / (1 + Domestic Rate)

Applying IRP to Our Example

Given:
‚Ä¢ Current spot rate: ‚Çπ90.00 per dollar
‚Ä¢ India interest rate: 4.00%
‚Ä¢ US interest rate: 2.00%
Calculate:
Expected Future Spot = 90.00 √ó (1.04) / (1.02)
Expected Future Spot = 90.00 √ó 1.0196
Expected Future Spot = ‚Çπ91.76 per dollar
üìâ Market Expects Rupee DEPRECIATION

Change: 91.76 - 90.00 = 1.76 rupees
Percentage: (1.76 / 90.00) √ó 100% = 1.96% depreciation

Why Does the Rupee Depreciate?

1
The Problem: If rupee DIDN'T depreciate, India offers 4% vs. US 2% ‚Üí Everyone invests in India forever!
2
Capital Floods In: Investors sell USD, buy INR ‚Üí Rupee appreciates TODAY
3
Market Prices: EXPECTED future depreciation offsets interest advantage
4
Equilibrium Restored: Total returns equalize ‚Üí No more arbitrage

üìê Part 8: The Mathematics

Appreciation Formula

Appreciation % = [(Old Rate - New Rate) / Old Rate] √ó 100%
Example: ‚Çπ92.00 ‚Üí ‚Çπ90.00
= [(92.00 - 90.00) / 92.00] √ó 100%
= [2.00 / 92.00] √ó 100%
= 2.17% APPRECIATION

Depreciation Formula

Depreciation % = [(New Rate - Old Rate) / Old Rate] √ó 100%
Example: ‚Çπ90.00 ‚Üí ‚Çπ91.76
= [(91.76 - 90.00) / 90.00] √ó 100%
= [1.76 / 90.00] √ó 100%
= 1.96% DEPRECIATION

Combined Return Formula

Total Return = (1 + Interest Rate) √ó (Starting FX / Ending FX) - 1
Interest: 4% = 0.04
Starting FX: ‚Çπ90.00/$
Ending FX: ‚Çπ91.76/$
= (1.04) √ó (90.00 / 91.76) - 1
= 1.04 √ó 0.9808 - 1
= 0.0200 = 2.00%
‚úì Matches US return! IRP confirmed!

üí° Part 9: The Three Core Insights

üéØ INSIGHT #1: Timing is Everything

  • Arbitrage exists for SECONDS, not DAYS
  • Winners: Algos and banks acting in first 30 seconds
  • Losers: Everyone else (retail investors)
  • The "2% advantage" exists only BEFORE spot rate adjusts
  • AFTER adjustment: advantage is GONE

‚ö° INSIGHT #2: Spot Rate Adjusts IMMEDIATELY

  • When rate differential changes ‚Üí Spot moves FIRST (minutes)
  • Expected future depreciation is PRICED IN today
  • Future spot rate adjusts GRADUALLY (months)
  • Appreciation = IMMEDIATE
  • Depreciation = EXPECTED (priced into today's rate)

‚öñÔ∏è INSIGHT #3: Returns Equalize in Equilibrium

  • Expected Return (Country A) = Expected Return (Country B)
  • Interest advantage = Expected FX loss
  • Higher interest rate ‚â† Free money
  • Higher interest = Compensation for expected depreciation
  • Markets are efficient ‚Üí No free lunch
